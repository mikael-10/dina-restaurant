# 🍽️ Dina Restaurant – Sito Web Responsive

Benvenuto nel progetto **Dina Restaurant**, un sito web completo e responsive sviluppato per un ristorante elegante.  
Il sito include più pagine (Home, Chi siamo, Menu, Team, Prenotazioni, Contatti) ed è ottimizzato per desktop e dispositivi mobili.

## 📁 Struttura del Progetto

- `index.html` – Homepage
- `about-us.html` – Chi siamo
- `menu-2-col.html` – Menu
- `team-2-cols.html` – Team
- `reservation.html` – Prenotazioni
- `contact.html` – Contatti
- `style.css` – Foglio di stile principale
- `/images/` – Cartella immagini
- `README.md` – Documentazione del progetto
